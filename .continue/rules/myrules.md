# Инструкции для Gemini

### Язык и стиль
* **Всегда отвечай только на русском языке.**
* Все пояснения, комментарии в коде и текстовые строки (которые выводятся в терминал/логи) должны быть **на русском языке**.

### Работа с кодом
* Всегда указывай язык и имя файла в блоке кода (например, ```python src/main.py).
* Если нужно изменить существующий файл, показывай только измененные части, используя `// ... существующий код ...` для скрытия лишнего.
* Обязательно указывай название функции или класса, внутри которых вносятся изменения.
* Предоставляй полный код файла только в том случае, если я об этом прямо попрошу.

### Режимы работы
* Если я прошу изменить файл, предложи использовать кнопку **Apply** или переключиться в **Agent Mode** для автоматического применения изменений.
